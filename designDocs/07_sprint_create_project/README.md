\# Sprint 07 — Création de projet



\## 🎯 Objectif

Permettre à l’utilisateur de créer un nouveau projet depuis l’interface frontend.



\## 📌 Contexte

Le système permet déjà :

\- d’afficher la liste des projets

\- de consulter le détail d’un projet

\- de suivre le pipeline et les insights



Mais il est impossible de créer un projet depuis l’interface.



\---



\## 🧩 Fonctionnalités à ajouter



\### 1. Bouton "New project"

\- rendre le bouton fonctionnel

\- ouvrir un formulaire ou modal



\### 2. Formulaire de création

Champs :

\- nom du projet (business idea)

\- catégorie (optionnel)



\### 3. Création projet

\- appel API backend

\- ou mock si endpoint absent



\### 4. Mise à jour UI

\- rafraîchir la liste après création

\- afficher le nouveau projet



\---



\## ⚙️ Contraintes

\- ne pas casser le design existant

\- garder une UX simple

\- utiliser TypeScript propre

\- code modulaire



\---



\## 📦 Résultat attendu

Un utilisateur peut créer un projet depuis l’interface et le voir apparaître immédiatement dans la liste.
